# Chef Table

Chef Table is a recipe assistant I built that takes ingredients you have at home and gives you recipes using AI. You can also share your own recipes with other users on the platform.

Live site: https://chef-bot-frontend.vercel.app

## What it does

- Add the ingredients you have and get two AI-generated recipes with step-by-step instructions
- Login with your Google account via Auth0
- Browse and create posts on the community recipes feed

---

## Built with

- React, Vite, CSS on the frontend
- Node.js and Express on the backend
- MongoDB for storing users and posts
- Groq API with LLaMA3 for AI recipe generation
- Auth0 for authentication
- Deployed on Vercel

---

## Running locally

Clone the repo and install dependencies in both backend-side and client-side-app folders. You will need a Groq API key, MongoDB URI, and Auth0 credentials in a .env file in the root folder.

Start the backend with `npm start` and the frontend with `npm run dev`.
